# 🎵 YouTube to MP3 Converter (Local Version)

เว็บแอปพลิเคชันสำหรับแปลงลิงก์ YouTube เป็นไฟล์ MP3 สำหรับดาวน์โหลดไว้ใช้งานส่วนตัว พร้อมระบบเติมเลขแทร็กนำหน้าชื่อไฟล์อัตโนมัติ เพื่อนำไปเสียบใช้งานเล่นเพลงเรียงลำดับบนเครื่องเสียงรถยนต์

## 💻 ความต้องการของระบบ (Prerequisites)
ก่อนเริ่มรันโปรเจกต์นี้ ต้องติดตั้งโปรแกรมเหล่านี้ในเครื่องก่อน:
1. **Python 3.x**
2. **FFmpeg** (จำเป็นสำหรับใช้แปลงไฟล์เสียงเป็น .mp3)
