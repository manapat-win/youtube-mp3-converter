# 🎵 YouTube to MP3 Converter (Local Version)

เว็บแอปพลิเคชันสำหรับดาวน์โหลดเพลงจาก YouTube และแปลงเป็นไฟล์ MP3 ทันที ออกแบบมาเพื่อการใช้งานส่วนตัว โดยมีจุดเด่นคือ **ระบบเติมเลขแทร็กนำหน้าชื่อไฟล์อัตโนมัติ** (เช่น `01_ชื่อเพลง.mp3`) เพื่อแก้ปัญหาเครื่องเสียงรถยนต์เล่นเพลงสลับคิว

## 💻 สิ่งที่ต้องเตรียมก่อนใช้งาน (Prerequisites)
สำหรับผู้ที่ไม่เคยเขียนโปรแกรมมาก่อน กรุณาทำตามขั้นตอนเหล่านี้เพื่อเตรียมคอมพิวเตอร์ของคุณ:

1. **ติดตั้ง Python:** ดาวน์โหลดตัวติดตั้งจากเว็บไซต์ Python (อย่าลืมติ๊กถูกที่กล่อง `Add python.exe to PATH` ก่อนกดติดตั้ง)
2. **ติดตั้ง FFmpeg:** ตัวโปรแกรมต้องการเครื่องมือนี้เพื่อแปลงไฟล์เสียง ให้ดาวน์โหลดไฟล์ `ffmpeg.exe` แล้วนำมาวางไว้ในโฟลเดอร์เดียวกับโปรเจกต์นี้
3. **ไฟล์ Cookies:** เพื่อป้องกันระบบบล็อกของ YouTube ให้ใช้ Chrome Extension ชื่อ `Get cookies.txt LOCALLY` โหลดคุกกี้ออกมา ตั้งชื่อว่า `www.youtube.com_cookies.txt` และวางคู่กับไฟล์หลัก
