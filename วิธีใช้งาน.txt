วิธีเข้าใช้งาน youtube to mp3 ของ manapat

1.ลง Python ในเครื่องให้เรียบร้อย

2.เปิด Terminal แล้วรันคำสั่งสั่งตั้งไลบรารี: pip install -r requirements.txt   

3.ติดตั้งโปรแกรม FFmpeg ลงในเครื่องของเขาเอง 
(เพราะคอมพิวเตอร์ทั่วไปจะไม่มี FFmpeg ติดมากับ Windows เหมือนบน Render 
ตัวโค้ด yt-dlp เลยจะแปลงไฟล์จาก .webm มาเป็น .mp3 ไม่สำเร็จหากไม่มีโปรแกรมนี้ติดตั้งอยู่ในระบบ Path ของเครื่อง)

4.ดึงไฟล์ Cookies จาก Chrome ของตัวเอง มาตั้งชื่อว่า www.youtube.com_cookies.txt แล้ววางคู่กับไฟล์ app.py 
เพื่อใช้บัญชีของตัวเขาเองในการคุยกับ YouTube ครับ
