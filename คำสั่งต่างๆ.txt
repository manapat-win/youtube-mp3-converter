git add . เอาไว้เลือกไฟล์ที่เราเพิ่งเเก้ไขทั้งหมด
git commit -m "test ver1" บันทึก(commit) พร้อมข้อความอธิบาย
git log --oneline เช็คว่า commit รึยัง

# สเตปที่ 1: เลือกไฟล์ทั้งหมดในโฟลเดอร์เตรียมเข้าคิวบันทึก
git add .

# สเตปที่ 2: บันทึกประวัติความเปลี่ยนแปลงลงในเครื่องตัวเอง (Local Commit)
git commit -m "พิมพ์ข้อความอธิบายสั้นๆ ตรงนี้ว่าเราแก้ไขอะไรไป"

# สเตปที่ 3: ดันโค้ดที่บันทึกไว้ในเครื่องขึ้นเว็บไซต์ GitHub (Remote Push)
git push origin main
