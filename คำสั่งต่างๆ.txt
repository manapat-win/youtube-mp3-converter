1. ตรวจสอบสถานะการทำงานปัจจุบัน
git status        (เช็คว่ามีไฟล์ไหนถูกแก้ หรือถูกบล็อกด้วย .gitignore บ้าง)
git log --oneline (เช็คประวัติการกดเซฟ (commit) ย้อนหลังแบบสั้นๆ)

2. สเตปการบันทึกและอัปโหลดโค้ดขึ้น GitHub
ขั้นที่ 1: เตรียมไฟล์ทั้งหมดเข้าคิว
git add .

ขั้นที่ 2: บันทึกความเปลี่ยนแปลงลงในคอมพิวเตอร์ (เซฟออฟไลน์)
git commit -m "อธิบายสิ่งที่แก้ไขไป เช่น แก้บักโหลดเพลง"

ขั้นที่ 3: ดันโค้ดที่เซฟไว้ขึ้นไปอัปเดตบนหน้าเว็บ GitHub
git push origin main
